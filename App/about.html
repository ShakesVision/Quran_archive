This app has been built with a focus on recreating the 15-Line IndoPak Mushaf, memorizing the Quran and quick search engine for the arabic text.

The content also aims to provide a line by line translation of the text which requires manually setting each line for the arabic text.

All the other apps provide either a continous text, or a version separated by ayahs.

Features
  1. Unicode: Everything is searchable and copyable, thanks to Unicode version
  2. Offline: No need to worry about the internet connectivity everytime.
  3. Optimized: Calculates everything using 1 file, saving bandwidth and memory. Update only when new changes are made.
  4. Quick Nav: Jump to Page, Juz, Surah, Ruku, Section (¼, ½, and ¾ of a Juz)
  5. Bookmarks: Auto bookmarks for scanned and unicode versions separately
  6. First searchable recreation Recreation of scanned version in completely digital format
  7. Hifz assist: Hifz mode, search, memorization features, Translation mode (in progress), everything you need to track your hifz progress
  8. Fast search: Advance and blazing-fast search, using Urdu/Arabic keyboard, with or without tashkeel
  9. Share everything: Search results copyable with pretiified output, each page can be exported with line breaks or as contineous text
  10. Listen to recitations: Expedite your hifz process by listening to recitations by 11 of the best Qurra'
  11. Add, Update, Delete, Import and Export memorization stat.
  12. Set foreground and background colors / image URL for customized look
  13. Highlight search result on mushaf
  14. Neat UI with material design
  15. Adjust font-size
  16. Memorization track under Juz page (highlighted in green)
  17. Motivational UI/UX for the hifz
  18. Scanned version for reference, with full navigation of Juz, Surah, Page, Section etc.
  19. Adjustable image quality for different internet connectivity
  20. Listen to audio on the fly, search surahs with english/arabic/persian/urdu name or number
  21. Download all data for free, always!
